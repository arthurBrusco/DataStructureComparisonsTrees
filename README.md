

Projeto desenvolvido para a disciplina de Estruturas de Dados (UFRGS) com o objetivo de comparar o desempenho de diferentes árvores binárias na resolução de um problema real.



##  Objetivo

Desenvolver uma aplicação capaz de estimar o tempo total necessário para jogar uma lista de jogos, utilizando dados reais da plataforma Steam, e comparar o desempenho de diferentes estruturas de dados.

---

##  Funcionamento

A aplicação recebe dois arquivos de entrada:

*   Arquivo CSV contendo:

  * Nome do jogo
  * Tempo médio de jogo (em horas)

*  Arquivo TXT contendo:

  * Lista de jogos desejados (um por linha)

---


##  Aprendizados

* Implementação prática de árvores binárias
* Comparação de desempenho entre estruturas
* Manipulação de arquivos em C
* Análise de complexidade
